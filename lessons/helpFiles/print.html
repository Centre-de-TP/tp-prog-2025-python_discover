The "print" Function

In programming, it will be very useful for you to display text or elements on the computer.

Each line of code is an order that you give to the computer.
So in order to display something, you will have to write a code line which tell the computer to display the desired element.

The magic thing to do that is the print.
But basically when you write your print, you tell the computer to display the thing between the parenthesis.
You can see an example below :

print("Hello World") >>> Hello World

Here you tell the computer to display the sentence "Hello World".

It's possible to write other things than sentences like in the example below :

print(1001) >>> 1001

Here you tell the computer to display the number 1001.

To go further with the print, it's possible to let the print create sentences with words you give.

print("I'm", "a", "sentence") >>> I'm a sentence

Here you tell the computer to display the sentence "I'm a sentence" after building it.
