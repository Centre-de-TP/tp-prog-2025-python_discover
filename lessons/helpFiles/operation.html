The "Operations" in python

Here is the time when Mathematics enter the programming language.

In python, there is a lot of operation which exists and allow the user to do a lot of things.
First, you should be familiar with the integer operations.

Name Operator Symbol Example Result
Addition + 6 + 2 8
Subtraction - 6 - 2 4
Product * 6 * 2 12
Division / 6 / 2 3

But now that you have seen the types, let's introduce some more complex operations.
First, the division may give you a floating number even if it's applied between two integers.
So in programming we designed other operations to prevent from that.

print(7 / 2) >>> 3.5
Name Operator Symbol Example Result
Division / 7 / 2 3.5
Integer Division // 7 // 2 3

To understand properly the integer division, you may have seen earlier in school when you where learning about the division, what is the rest in a division.
Until now, you always put the rest after the "," in your result.
But sometimes you may don't want the rest of the division, then you keep only the integer part of the result, the one before the ",".

Also, in order to complete some operation more quickly, you will find more complexe operations.

Name Operator Symbol Example Result
Power ** 7 ** 2 49
Modulo % 7 % 2 1

Here you can see a really complexe operation called the "modulo".
The concept of this operation is similar to the concept of integer division.
When, in the integer division you keep the integer part of a result, in the opposite, when you use the modulo you keep the left part of the integer division.

To re-explain the previous example, when we decompose the operation "7 % 2", it gives us "(2 * 3 + 1) % 2".
This is also the same as keeping every module of 2 in the parenthesis out and leaving the rest.
It then gives us "1" left and "2 * 3 / 2" which is an integer.
So the result is 1.
