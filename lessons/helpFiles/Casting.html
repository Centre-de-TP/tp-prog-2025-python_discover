Python - Help - Casting

The "casts" functions

Now that you saw the types, it's often useful to change an object type.
To do this, we created some other functions that a program can call.

variable = 42 print(type(variable)) >>> int old_variable = 42 new_variable = str(old_variable) print(type(new_variable)) >>> str

As you may have noticed already, to change an object type, you just need to call the function which name is the name of the desired type.

variable = "42" print(type(variable)) >>> str old_variable = "42" new_variable = int(old_variable) print(type(new_variable)) >>> int

But be careful with the type changing.
Try to create an int with a text containing other things than numbers will create an error.

old_variable = "text" new_variable = int(old_variable) print(type(new_variable)) >>> ValueError: invalid literal for int() with base 10: 'text'
