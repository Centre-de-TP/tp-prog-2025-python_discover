The "input" Function

When you do a program, you may have to ask the user for information (like a searched music for example).

Remember that you need to tell the computer to ask the user an information.

The magic thing to do that is the input function.
To make it simple, when you use your input function, you tell the computer to display the thing between the parenthesis to the user and to send you the information that the user gives you.
You can see an example below :

input("What's your favorite music?") >>>

Here you tell the computer to display the sentence "Ask the user for it favorite music".

It's possible to ok other things than sentences like in the example below :

input(1001) >>>

Here you tell the computer to ask the user "1001".
But this thing doesn't have any sens, this is why it's not used.

In difference with the print, an input function accepts only one element between the parenthesis.

input("Tell", "me", "something") >>> Error: TypeError: input() takes at most 1 arguments (3 given) on line 1

To understand the error, the thing you write between the parenthesis are called arguments.

Before ending the lesson, you can use the result given by the user if you want to.
As shown in the next exemple, you just have to use the piece of code like if it already was the user response.

print(input("Tell me your favorite music.")) >>> Believers from Imagine Dragons
