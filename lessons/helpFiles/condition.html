The condition checker

But you will ask "Why these things can be used for ?".
It's the time to introduce the conditions.
In programming, it could be a great thing to check if a value is correct before engaging another action.

Example :
Imagine a situation where I have a number and if the number is odd, I want to write "odd", otherwise I should write "even".
To check if a number is odd, we have seen the modulo earlier which will help us a lot.

Here is an example :

number = int(input("Enter a number: ")) print("The chosen number is " + str(number)) if (number % 2 == 0) : print("The number is even") else : print("The number is odd")

In the previous example we used two very important thing in python.

The first one is the use of a condition checker.
This will execute the actions within a given "scope" if the condition is true.
Two checker were used here.
The first one is an "if" statement which will execute the scope if the condition is true.
The second is an "else" statement which will execute the scope if the previous "if" condition isn't true.

The second very important thing is the scope definition.
In python when an action depend on a previous line, it's mandatory to add a space right after it.
In the previous example, the print("The number is even") depend on the if statement.
This is used when the previous line ends with the symbol ":".

You will use some different condition checker from now one.
Let's see them :

Statement Name Construction Explanation
if statement if (condition) : If the condition between the parenthesis is valid, the scope is executed.
elif statement elif (condition) : If the previous "elif" and "if" 's condition weren't True and the condition between the parenthesis is valid, the scope is executed.
else statement else : If the previous "elif" and "if" 's condition weren't True, the scope is executed.


More complex conditions

Because we don't want the dev to write a lot of "if" statement when there is a complex condition to check, we have added a very useful thing to the language.*
It's easy to say when a human is talking to another human that :
"he did something, and he did another thing".

In programming, it's the same.

Name Operator Symbol Utilisation Explanation
AND and condition1 and condition2 Both condition need to be True so the resulting one could be True.
OR or condition1 or condition2 At least one of the condition need to be True so the result is True.
NOT not not condition The condition needs to be False so the result is True.

If you want to go further, it's also possible to add parenthesis to create very complex conditions.
But for now let's have an example of these new statements.

number = int(input("Enter a number: ")) number2 = int(input("Enter a second number: ")) print("The chosen numbers are : " + str(number) + " and " + str(number2)) sum = number + number2 print("The sum is : " + str(sum)) if (2 * number == sum and 2 * number2 == sum) : print("The sum is the double of both numbers !") elif (2 * number == sum or 2 * number2 == sum) : print("The sum is the double of one number only") elif (not number == sum) : print("The sum which isn't the double of number, isn't equals to it either.")
